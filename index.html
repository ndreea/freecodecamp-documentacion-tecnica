Introducción

Hojas de Estilo en Cascada (del inglés Cascading Style Sheets) o CSS es el lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML (incluyendo varios lenguajes basados en XML como SVG, MathML o XHTML). CSS describe cómo debe ser renderizado el elemento estructurado en la pantalla, en papel, en el habla o en otros medios.

Selectores

Un selector es un conjunto de reglas de estilos aplicadas múltiples elementos de la hoja html. En lugar de aplicar el estilo elemento por elemento, lo aplicamos de manera global.

Principales tipos de Selectores

Id

Se define mediante una # justo antes del nombre de la id. Las reglas de estilo que contiene se aplican al elemento que tenga la id indicada.

#id{
/* Estilos */
}

Clase

Se define mediante un . justo antes del nombre de la clase. Las reglas de estilo que contiene se aplican a cualquier elemento que tenga la clase indicada. Debido a que un elemento puede tener varias clases, también puede recibir estilo de varios selectores de clase.

.clase{
/* Estilos */
}

Elemento

Aplica estilo a todos los elementos que tengan la etiqueta html indicada. Por ejemplo, un selector del elemento div aplicaría estilos a todos los div.

div{
/* Estilos */
}
  • En caso de contradicción entre selectores, se aplicarán esta jerarquía de prioridad: !important > inline style > ids > clases > elementos.
  • En caso de igual importancia entre selectores que se contradicen, los últimos en haber sido escritos tendrán prioridad.
  • La aplicación de estilos inline se penaliza en SEO.

Variables

Declaración y asignación: --nombrevariable: value;

Utilización: propiedad: var(--nombrevariable, fallback);

  • El fallback no es necesario. Se utiliza para que el navegador tenga un respaldo en caso de que la variable sea inválida. Es útil para debuggear.
  • Otra manera de hacer respaldos es poner la misma propiedad justo antes con un valor con más soporte (aprovechando que, en caso de igual importancia, la prioridad va de abajo hacia arriba).

Las variables en CSS también tienen efecto cascada. Por tanto, son utilizables dentro del elemento en el que han sido creadas y de cualquiera de los elementos que estén anidados dentro. Por esto, las variables CSS suelen declararse en la pseudoclase :root (contenedor de todo el documento .html).

Las variables se pueden sobreescribir volviéndolas a declarar dentro de elementos más específicos que aquel en el que fueran declaradas por primera vez.

Media Queries

Permiten cambiar los valores de las reglas de un selector en función del tamaño del viewport. Se utilizan para hacer contenido adaptativo. También pueden alterar el valor de las variables.

@media (max-width: 350px) {
:root {
--penguin-size: 200px;
}
p {
color: red;
}
}

Esta media querie se activaría si el ancho del viewport es menor o igual a 350 píxeles.
